# T-Shirt Point

Full-stack T-Shirt e-commerce app.

## Structure
- `Frontend/` - React + Vite + Tailwind (Port 3000)
- `Admin/` - React + Vite + Tailwind (Port 3001)
- `Backend/` - Node.js + Express + MongoDB (Port 5000)

## Setup

### Backend
```bash
cd Backend
npm install
# Add your MONGO_URI in .env
npm run dev
```

### Frontend
```bash
cd Frontend
npm install
npm run dev
```

### Admin
```bash
cd Admin
npm install
npm run dev
```
